# QUBE's FIRST Tech Challenge Robot Controller App
This repository contains the code for "Mihai Viteazul" National College's QUBE FTC team.

## Build instructions
- You will need to install / update Android Studio's [Kotlin](https://kotlinlang.org/) plugin.
- You will need [Git LFS](https://git-lfs.github.com/) to download the binary files included in this repository.
  After using `git clone`, also run `git lfs install` and `git lfs pull` to download all the files.
